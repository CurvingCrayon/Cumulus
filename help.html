User Manual

Introduction

Cumulus Note Consolidator allows for integration of documents and files from the three most popular cloud storage services: Google Drive, Dropbox, and OneDrive, as well as the integration of the user's local files. Each and any of these services can be logged in and out of at any time the user pleases, though it should be noted that the user will not be able to access files from a service they have logged out of. Opening a file consists of the following steps:

  1. Logging in to the appropriate service
  2. Opening a file

Logging In

To log into a cloud storage service through Cumulus, hover over the tab that contains the associated cloud storage service icon. The following is a guide to which icon matches which cloud storage service:

  • Google Drive
  • Dropbox
  • OneDrive
Hovering over the chosen tab will cause it to drop down, revealing a "Login" button. If the "Logout" and "Open File" buttons are seen, this means you are already logged in. To log in, click the "Login" button. This will cause a login dialog to appear on this screen. You may notice that this popup is from the official website of the cloud storage service, meaning that logging in is secured with HTTPS. Cumulus never sees any of your account passwords - they are passed straight to the cloud storage services. If you do not have an account for the associated cloud storage service, an account can be created through the dialog by following the appropriate links. If your login details are correct, and the login is successful, the dialog will close automatically after a small loading time, and the associated tab will be outlined in blue to indicate that the service has successfully been logged into.

Opening Files

To open a file from a cloud storage service, hover the cursor over the associated cloud storage service tab. This will cause the tab to drop down, from which the "Open File" button can be clicked. If the "Open File/s" button cannot be found, the service must be logged in first. Clicking the "Open File/s" button will cause the "Select File/s" dialog to appear, from which a file, or multiple files, can be selected to open.

Selecting

To select a file, click on the file name or image. This will cause the file to be highlighted orange. To select multiple files, a single file can be clicked, then additional files can be selected by holding the "control" key in Windows, or the "command" key on Mac whilst clicking. A group of consecutive files can be selected by clicking on the first file, then holding the "shift" whilst clicking on the last file in the group. Clicking on blank space inside the dialog will deselect all files. These controls are very similar to those used in Windows' "File Explorer", and OSX' "Finder" programs. If a folder is selected, all the files contained in that folder will also be selected.

Searching

The search bar can be used to narrow down file options by clicking and typing a search query and hitting the "enter" key. Any files with names containing the search query will be displayed. To revert back to viewing all files, remove the search query from the search bar and hit the "enter" key.

Navigating

Folders within cloud storage services will appear using a folder icon. Double-clicking on it will open the folder and show the contents of it. The folder path, also known as breadcrumbs, will appear at the top of the dialog. Entering folders within folders will increase the length of this breadcrumb trail, but individual breadcrumbs can be clicked to quickly navigate to a specific parent folder. The "go up a level" button can also be clicked to go up to the parent of the currently opened folder.
